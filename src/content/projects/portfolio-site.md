Built with Astro, styled with Tailwind, written in Obsidian, deployed to Cloudflare Workers.

### Key Features

- **Dark Theme**: Minimalist, terminal-inspired design.
- **Obsidian CMS**: The `src/content` folder is an Obsidian vault.
- **YouTube Integration**: Automatically fetches latest videos.
- **Cloudflare Ready**: Optimized for Cloudflare Workers.
